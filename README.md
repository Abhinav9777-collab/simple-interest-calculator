# Simple Interest Calculator

This project is a simple interest calculator created as part of a software engineering assignment.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Inputs
- Principal Amount
- Rate of Interest
- Time (in years)

## Output
- Calculated Simple Interest

## Example
If:
- Principal = 1000
- Rate = 5
- Time = 2  

Output:
- Simple Interest = 100
